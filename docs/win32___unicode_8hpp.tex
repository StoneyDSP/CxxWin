The header file provides the generic function name implemented as a macro. 

The preprocessor expands the macro into either the Windows code page or Unicode function name. The letter \"{}\+A\"{} (ANSI) or \"{}\+W\"{} (Unicode) is added at the end of the generic function name, as appropriate. The header file then provides two specific prototypes, one for Windows code pages and one for Unicode.

\#if defined(\+UNICODE) \#define Set\+Tst Set\+TstW \#else \#define Set\+Tst Set\+TstA \#endif

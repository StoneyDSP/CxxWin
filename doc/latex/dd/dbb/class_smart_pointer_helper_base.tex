Query a COM object for a pointer to one its interfaces, identifying the interface by a reference to its interface identifier (IID). 

If the COM object implements the interface, then it returns a pointer to that interface after calling \textquotesingle{}\doxylink{class_smart_pointer_helper_base_a2c9426410eeb136beba8123b5d84dc6e}{Smart\+Pointer\+Helper\+Base\+::\+Add\+Ref()}\textquotesingle{} on it.


\begin{DoxyParams}{Parameters}
{\em riid} & A reference to the identifier of the interface being queried for.\\
\hline
{\em ppv\+Object} & The address of a pointer to an interface with the IID specified in the \textquotesingle{}riid\textquotesingle{} parameter. Because you pass the address of an interface pointer, the method can overwrite that address with the pointer to the interface being queried for. Upon successful return, \texorpdfstring{$\ast$}{*}ppv\+Object (the dereferenced address) contains a pointer to the requested interface. If the object doesn\textquotesingle{}t support the interface, the method sets \texorpdfstring{$\ast$}{*}ppv\+Object (the dereferenced address) to nullptr.\\
\hline
\end{DoxyParams}
